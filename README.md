# ☕ Java GUI Applications

## 📌 About The Projects
This repository contains a collection of interactive desktop applications developed using **Java** and the **Swing** framework. These projects serve as a practical demonstration of my skills in **Object-Oriented Programming (OOP)**, event-handling, and designing user-friendly Graphical User Interfaces (GUI).

## 🚀 Included Applications

### 1. ✖️⭕ Tic-Tac-Toe Game
A fully functional, classic 2-player Tic-Tac-Toe game. 
* **Features:** Real-time score tracking for Player X and Player O, win/draw detection, and a clean interface built with NetBeans GUI builder.

### 2. 🧮 Smart Calculator
A reliable desktop calculator application.
* **Features:** Handles core arithmetic operations efficiently with a responsive layout and clear display, mimicking standard OS calculators.

## 🛠️ Technologies & Tools Used
* **Programming Language:** Java
* **UI Framework:** Java Swing / AWT
* **Architecture:** Core OOP concepts
* **IDE:** Apache NetBeans
